Trigger builds on merge/pull request events.

NOTE: This option requires additional git scm configuration for the job:
  • 1. Goto job Configure page
  • 2. Goto Source Code Management
  • 3. Select Git, under Advanced, set Name to origin and:
    • If the project is private, set refspec to +refs/heads/*:refs/remotes/origin/* +refs/merge/*/MERGE:refs/remotes/origin/merge/*
    • If the project is public, set refspec to +refs/heads/*:refs/remotes/origin/* +refs/pull/*/MERGE:refs/remotes/origin/pull/*
  • 4. Click Apply
